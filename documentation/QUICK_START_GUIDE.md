## ğŸ› ï¸ **TROUBLESHOOTING**

### **Common Issues:**

**1. Camera Not Working:**
- Check camera permissions
- Ensure no other applications are using the camera
- Try restarting the application

**2. Model Training Fails:**
- Ensure dataset is properly uploaded
- Check available memory (requires ~2GB RAM)
- Verify TensorFlow installation

**3. Voice Commands Not Working:**
- Check microphone permissions
- Install required packages: `pip install SpeechRecognition pyaudio`
- Restart application after installation

**4. Low Recognition Accuracy:**
- Use clear, well-lit iris images
- Ensure proper camera focus
- Retrain model with more diverse dataset
